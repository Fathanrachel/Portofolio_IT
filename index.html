About Me

Halo! Saya Muhammad Rachel Fathan Idzhany, mahasiswa semester empat Teknik Informatika di Universitas Pancasila. Saya memiliki passion dalam pengembangan front-end, dengan keahlian dalam HTML, CSS, JavaScript, C++, C, SQL databse, java, python serta framework seperti Bootstrap.

Selama studi, saya telah mengerjakan berbagai proyek, mulai dari situs deskripsi film hingga sistem manajemen rental mobil, baik secara mandiri maupun dalam tim. Proyek-proyek ini telah membantu saya mengembangkan keterampilan teknis dan pemahaman.

Sebagai Mahasiswa Teknik Informatika, saya ikut mengorganisir seminar, serta menjadi mentor bagi mahasiswa baru dalam event mencari ide inovatif dan juga kreatif.

Saya selalu bersemangat untuk belajar dan mengikuti tren terbaru dalam front-end development, dengan tujuan terus meningkatkan keterampilan dan berkontribusi pada proyek-proyek inovatif yang meningkatkan pengalaman pengguna di web.
